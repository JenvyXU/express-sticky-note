# 在线便签
1. 克隆仓库到本地电脑
```
git clone git@github.com:jenvyxu/express-sticky-note.git
```
2. 进入目录并且安装npm包
```
cd express-sticky-note
npm i
```
3. 运行node服务器
```
npm run start
```
4. 打开浏览器输入https://localhost:4000
